## 📱 支持的设备列表

本固件基于 **ImmortalWrt（padavanonly / mt798x-6.6）** 源码编译，针对不同 MediaTek MT798x 设备进行适配优化，提供稳定可靠的 OpenWrt 使用体验。

| 设备名称 | 芯片平台 | 硬件定位 | 主要特点 / 适用场景 |
|----------|----------|----------|----------------------|
| **CLX_S20P** | MT7986 | Wi-Fi 6 入门级 SoC | 家庭主路由 / 旁路由基础使用，功耗低，适合长期稳定运行 |
| **Cetron_CT3003** | MT7981 | Wi-Fi 6 AX3000 级 | 中小户型无线覆盖设备，支持 Mesh 扩展，适合家庭主路由 |
| **Cudy_TR3000-V1** | MT7981 | Wi-Fi 6 AX3000 + 2.5G | 带 2.5G 口的高性价比设备，适合宽带升级/主路由 |
| **Cudy_TR3000-V1 (256MB)** | MT7981 | 低内存版本 | 适合轻量 OpenWrt 使用，仅基础路由/转发功能 |
| **JDCloud_RE-CP-03** | MT7986 | 高性能 Wi-Fi 6 SoC | 四核高性能平台，适合高并发/NAT/大流量转发 |
| **Netcore_N60-PRO** | MT7981 | Wi-Fi 6 家用级 | 稳定性优先，适合家庭多设备连接与基础插件环境 |
| **Philips_HY3000** | MT7981 | Wi-Fi 6 千兆级 | 偏稳定固件使用场景，适合长期运行的主路由 |
| **Qihoo_360T7** | MT7981 | Wi-Fi 6 AX3000 入门爆款 | 性价比高，适合入门 OpenWrt / 旁路由 / 家用主路由 |
